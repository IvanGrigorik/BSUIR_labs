ЛАБОРАТОРНАЯ РАБОТА №8

 

Цель: получить практические навыки мониторинга трафика компьютерной сети.

Лабораторная работа выполняется студентом индивидуально; на базе ПК, подключенного к компьютерной сети; по варианту.

Задание на лабораторную работу может содержать следующие пункты:

1. Установить программу Wireshark на ПК под управлением ОС Windows или Linux.

2. Изучить интерфейс Wireshark.

3. Несколько раз настроить Wireshark для отслеживания специально генерируемого трафика по нескольким критериям (согласно варианту).

4. Изучить синтаксис команды tcpdump (в Windows становится доступна после установки Wireshark).

5. Запуская tcpdump столько раз, сколько нужно, добиться отслеживания трафика по нескольким критериям (согласно варианту заданию).

6. По результатам работы составить отчет согласно установленной форме. Отчет может содержать скриншоты.

Основным вспомогательным материалом для выполнения данной лабораторной работы являются руководство пользователя программы Wireshark и описание команды tcpdump.
