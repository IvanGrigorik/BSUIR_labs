ПРАВИЛА ПРОВЕДЕНИЯ ЭКЗАМЕНА

 

Экзамен проводится в электронном виде с использованием специально разработанного на кафедре программного обеспечения.

Предусмотрены вопросы двух видов:
1. MCSA (Multiple Choice Single Answer) -- нужно выбрать один правильный вариант из нескольких предложенных (по схеме один из пяти).
2. MCMA (Multiple Choice Multiple Answer) -- нужно выбрать несколько правильных вариантов из всех предложенных (по схеме два из шести) - учитывается каждый правильный подответ.

На экзамене задаются по два вопроса (MCSA и MCMA) из каждого рассмотренного на лекциях раздела.

За правильный овет на вопрос MCSA дается два балла. За правильный подответ на вопрос MCMA дается один балл.

Оценка выставляется автоматически пропорционально набранному количеству баллов.

Пример вопроса MCSA:

На рисунке показан фрагмент взаимодействия по протоколу TCP. Чему равно поле AN?
( ) 101
( ) 102
( ) 201
( ) 202
( ) Ни один из перечисленных вариантов

Пример вопроса MCMA:

Дано поле Галуа GF(5). Какие операции из нижеперечисленных будут соответствовать этому полю? (Выберите два варианта)
[ ] 0 + 1 = 1
[ ] 2 + 3 = 5
[ ] 4 + 4 = 3
[ ] 0 * 1 = 1
[ ] 3 * 3 = 3
[ ] 5 * 5 = 0
